# Solana Rent Reclaimer Pro

Scan your Solana wallet for empty SPL token accounts and see how much SOL you can reclaim. Non-custodial, read-only, local. No keys, no risk.

## Usage
1. Paste your Solana wallet address.
2. Click "Scan Wallet".
3. View empty token accounts and estimated reclaimable SOL.
